## 🚀 Download & Install (5 minutes)

### 📦 Option 1: Professional Installer (Recommended)
**Download**: [NLight_IDE_3.0.4_Setup.exe](https://github.com/ishanthasiribaddana/nlight/releases/download/v3.0.4/NLight_IDE_3.0.4_Setup.exe)

The installer automatically downloads and sets up the full IDE (~240 MB). Internet connection required during installation.

### 🎯 Installation Steps
1. **Download**: Click the installer link above (76 KB)
2. **Run**: Double-click `NLight_IDE_3.0.4_Setup.exe`
3. **Follow**: The 4-step setup wizard
4. **Launch**: "NLight IDE" from the desktop shortcut
5. **Start coding**: Look for "NLight" in left sidebar → "Chat"

### 🔄 Option 2: Update Existing Installation
Already have NLight IDE? Just download the updater:

**Download**: [NLight_Update.exe](https://github.com/ishanthasiribaddana/nlight/releases/download/v3.0.4/NLight_Update.exe)

### 📦 Option 3: Manual Setup (Advanced)
Download the complete IDE package directly:

**Download**: [NLight-IDE-win32-x64-3.0.4.zip](https://github.com/ishanthasiribaddana/nlight/releases/download/v3.0.4/NLight-IDE-win32-x64-3.0.4.zip) (~240 MB)

Extract the ZIP and run `NLight.exe` inside the extracted folder.
